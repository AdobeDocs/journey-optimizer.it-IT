# isEmpty {#isEmpty}

Restituisce true se la stringa nel parametro non ha carattere.

## Categoria

Stringa

## Sintassi della funzione

`isEmpty(<parameters>)`

## Parametri

* stringa

## Firma e tipo restituito

`isEmpty(<string>)`

Restituisce un valore booleano.

## Esempio

`isEmpty("")`

Restituisce true.

`isEmpty("Hello World")`

Restituisce false.
