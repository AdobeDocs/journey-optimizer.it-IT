# trim {#trim}

Rimuove gli spazi iniziali e finali.

## Categoria

Stringa

## Sintassi della funzione

`trim(<parameters>)`

## Parametro

| Parametro | Tipo |
|-----------|------------------|
| string | string |

## Firma e tipo restituito

`trim(<string>)`

Restituisce una stringa.

## Esempio

`trim(" Hello ")`

Restituisce &quot;Hello&quot;.
